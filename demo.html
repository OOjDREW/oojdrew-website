Earlier Online Demo Requirements

Java version 1.4.0 or above with Java Web Start for version 0.88

There were some issues in OO jDREW version 0.88 with the web start applications and Java 1.4.0 releases under windows; where entering a large knowledge base into the text fields in application may cause the program/computer to freeze up. This appears to be an issue with specific JRE releases. If you experience this problem it is suggested that you upgrade to the latest 1.4.2 release of the JDK/JRE (the 1.5 series is currently untested but may also work).

Moreover, the downloaded version do not have these issues, and permit for larger knowledge bases, too.

Some features of OO jDREW cannot be used on the Java Web start version. This includes opening a file on a local file system (version 0.93+) as well as opening a file from a web source (version 0.93+). If you want to use these features you must use the downloaded version of OO jDREW.

Version 0.93+ of OO jDREW requires 1.6.0 of the java runtime environment. You can download the newest runtime enviroment of java from here.
