Requirements

Java version 1.4 or above with Java Web Start

There have been some issues with the web start applications and Java 1.4.0 releases under windows; where entering a large knowledge base into the text fields in application may cause the program/computer to freeze up. This appears to be an issue with specific JRE releases. If you experience this problem it is suggested that you upgrade to the latest 1.4.2 release of the JDK/JRE (the 1.5 series is currently untested but may also work).

Moreover, the downloaded version do not have these issues, and permit for larger knowledge bases, too.
